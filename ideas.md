# LGO メディアサイト デザインコンセプト案

## 選定基準
- ターゲット: 大学生（20代前半）の合宿幹事
- 目的: SEO集客 → LINE診断への送客
- ベンチマーク: モゲチェック（情報量×診断誘導の両立）

<idea>
**Design Movement:** Japanese Editorial × Digital-Native（日本の情報誌×デジタルネイティブ）

**Core Principles:**
1. 情報密度と読みやすさの両立（モゲチェック型のカラム構成）
2. LINEグリーンを主軸にした一貫したCTA色
3. 「幹事の悩み解決」を軸に、ファクトと数字で信頼を構築
4. スクロール追従型の強烈なCTA設計

**Color Philosophy:**
- Primary: LINEグリーン `#06C755`（CTAの絶対的な色として統一）
- Background: オフホワイト `#F7F8FA`（清潔感・読みやすさ）
- Accent: ディープネイビー `#1A2B4A`（見出し・信頼感）
- Muted: ライトグレー `#E8ECF0`（カード背景・区切り）

**Layout Paradigm:**
- 左サイドバー固定型（記事ページ）+ 中央コンテンツ + 右スティッキーCTAカラム
- トップページはフルワイドヒーロー → カード型コンテンツグリッド

**Signature Elements:**
1. 緑の太いアンダーライン装飾（見出し強調）
2. ランキングバッジ（1位:金、2位:銀、3位:銅）
3. 「寸止め」グレーアウトボックス（空き状況・正確見積もりの非公開エリア）

**Interaction Philosophy:**
- スクロール量に応じてサイドバーCTAが強調される
- ホバー時にカードが軽く浮き上がる（shadow強調）
- CTAボタンは微細なパルスアニメーション

**Animation:**
- ページ入場: フェードアップ（0.3s ease-out）
- カードホバー: translateY(-4px) + shadow強化
- CTAボタン: 2秒周期のpulseアニメーション（緑のglow）
- スクロール追従バナー: 下からスライドイン（初回のみ）

**Typography System:**
- 見出し: Noto Sans JP Bold（日本語の視認性重視）
- 本文: Noto Sans JP Regular（読みやすさ優先）
- 数字・強調: Oswald Bold（欧文数字のインパクト）
- 階層: H1=32px, H2=24px, H3=18px, Body=15px
</idea>
